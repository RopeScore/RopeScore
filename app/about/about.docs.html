1. The Dashboard

The above image shows a dashboard with multiple added categories.

The upper gray area is the menu, on the dashboard only "Dashboard" and "New Category" are avilable.

The first row of buttons gives you various tools manage your categories.

  • New: This button will create a new category
  • Import: This button will allow you to choose an exportfile from ropescore, saved on your computer, and REPLACE ALL categories with those in the export file. Make sure to export your current categories before importing new ones
  • Export: This button will export all your data to a file on your computer, you will be prompted to select where the file should be saved
  • Reset: This button resets the program, it removes ALL saved data. It's highly reccomended to do an export before resetting
  • Display Live Unofficial Scores: This will take you to a page thatcan be displayed for instance on an external projector, it will show unofficial scores, live when they are set to be displayed in the program
  • Combine Two Categories: Allows you to combine multiple categories into one
  • Reset Category Order: Resets the order the categories are displayed on the dashboard to be ordered by creation date, oldest first

The row of buttons below "Dashboard" (here "Masters", "Teams" etc.) will, once clicked, open the category view for that particular category. The hexadecimal number below them are their checksum, more info on this is avilable under Category

You can click the arrows (‹ and ›) on the sides of every category to sort your Dashboard

2. Configuration

The configuration will open either when you create a new category from the menu or the dashboard, or when you click configure in the Category view

Changing the configuration after scores have been entered may give unexpected results

2.1 Category Data

This is the configuration for a new category, in the Basics menu you must fill in a name for your category.

Simplified Levels will be described later.

Under Events you can select which events the category contains of by clicking the checkboxes to the right of their names. If an event is selected you must also specify how many judges that will judge the event.

If Simplified Leves are selected, the scores will be calculated a bit different, this is detailed under Result, the option to choose what factor the scores and ranks will be multiplied by to determine the final rank and score. If one of the fields is left blank the rank factors defined in fisac-irsf's judging manual will be used, and if it's an event that is not defined in fisac-irsf's judging manual and the field is left empty, a factor of one (1) will be used.

2.2 Participants

Upon clicking "Save and Continue" on the Category Data Configuration participants can be added, you have the option to select what id the participants will start with, this id is used for internal purposes, but is displayed as a tool for the Competition organiser. The id of a participant will never change, however the participant's name and club may, thus the id of the participant can be relied onto for identification.

Once "Add Participants" has been clicked a table where the participants name and club can be entered will show, name and club can be omitted for every participant as the id is used by the program.

Participants can also be imported from excel, or similar spreadsheet software. See import-examples.xlsx  for examples. Note that the header id will be ignored, this is to ensure that ther won't be any conflicts. the data pasted into the input area will be treated as tab separated content (tsv) and every value must be separated by a tab character.

3. Category

After adding participants to a category, or when the category is clicked on The Dashboard the Category View will be shown. This view gives an overview over the participants and the events, clicking "edit" in the column of an event on the row of a participant will open Score Entering.

Edit multiple is avilable for Speed events, where you can enter multiple speed scores at once

The checksums is a value calculated from the data entered, it can be compared to a second instance of RopeScore, if the two values match, then the entered data is equal

  • Overall: (green) The checksum in the top right corner of the category view grid shows the checksum for the whole category, this is also the number displayed on the Dashboard
  • Participant: (red) The checksum in the rightmost cell of every row shows the checksum of all data entered for that participant
  • Event: (blue) The checksum right to an Edit button shows the checksum for that participant's score in that specific event

4. Score Entering

The view to enter scores does differ for speed events and freestyle events

For all numerical inputs both decimal point (.) and decimal comma (,) are allowed

4.1 Speed

4.1.1 Single

Enter the scores each speed judge have turned in to their respective box, if the speed judges have signaled that the skipper made a false start that box should be checked. For team speed, false switches is avilable, if the judges have marked any false switches, the appropriate number of false switches should be entered. The last row shows the unofficial score, calculated on the fly

4.1.2 Multiple

Scores are entered the same way as for a single speed entry, however multiple participant's scores can be entered at once. The second-to-rightmost column shows the unofficial scores, calculated on the fly. pressing Display in the rightmost column will add that participant's speed score to the currently displayed speed events, see Live Display. Clicking display all will add all speed scores from the current event to the currently displayed speed events.

4.2 Freestyle

A judges are Presentation Judges, B judges are Required Elements Judges, D judges are Difficulty Judges, H judges are Head judges, and M judges are Miss judges. The task of judging misses might be delegated to other judges, if so their miss counts are entered as a Miss judge's score

The program will correct numbers greater than the allowed amximum, and indicate sow by adding a yellow border to the input field, for five (5) seconds. If it can't automatically correct it, a red border will be added to the input field and saving won't be allowed.

it is highly reccomended that zero (0) is entered instead of leaving a field blank

The bottom row calculates and displays Unofficial scores on the fly.

Pressing Display will display that participant's score only on the live display, see Live Display

5. Results

Results and ranking are generated for every event where a participant have complete scores entered. If all the events of a participant have complete scores entered the total score, rank sum and overall rank will be calculated as well. the overall score is calculated using the scores for the individual events multiplied by certain factors, see Category Configuration

The Results can be exported to excel by pressing the "Export to Excel" button.

The "Update Live" will recalculate the scores every five (5) seconds until disabled

If simplified levels are selected in the category's Configuration  the difficluties awards higher points per trick and the factors that the ranks and scores are multiplied against may differ from fisac-irsf's judging manual, as set in the category Configuration.
